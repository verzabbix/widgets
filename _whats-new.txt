02 - Simple Map
---------------

- Widget can be written purely in JavaScript.
- Widget is fully encapsulated as a class instance.
- Custom JavaScript class must be mentioned in the manifest file.
- Base JavaScript classes provide convenience.
- Base JavaScript classes define the default functionality.
- Widget is controlled by the dashboard framework.
- Widget is updated on demand by the framework.
- Promise-based visualization.
- Assets must be defined in the manifest file.
